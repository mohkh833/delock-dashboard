# Eyris - React Admin Template

A modern React admin template built for real production apps, not just demos. It combines high performance with a rich collection of handcrafted UI components, all built natively into the codebase so you're never locked into external design systems or libraries.

Eyris is designed with professional, production-oriented architecture, giving you a clean project structure that scales as your application grows. It mirrors how real SaaS and enterprise dashboards are built, making it easy for teams to extend, maintain, and onboard new developers.

Whether you're building a SaaS platform or an internal dashboard, Eyris delivers the right balance of aesthetics, flexibility, and developer experience. You get light and dark mode, RTL support, multi-language support, various layout options, powerful chart widgets, clear documentation, and MCP server integration — enabling AI agents to understand and reuse components directly in your project.

## Features

- **Tailwind Component-Based Architecture**: Reusable components to streamline your development process.
- **TypeScript** - Full type safety throughout
- **Next.js** - Production-ready React framework with App Router
- **Tailwind CSS 4** - Modern utility-first styling
- **Dark Mode** - Built-in theme support
- **i18n Ready** - Multi-language support
- **Auth System** - Authentication and authorization built-in
- **AI-Powered Development** - MCP server for AI assistant integration

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
